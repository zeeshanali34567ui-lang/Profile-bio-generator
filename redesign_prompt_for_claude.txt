--------------------------------------------------------------------------------
PART 3: CSS & STYLING SPECIFICATIONS
--------------------------------------------------------------------------------
* Framework: Tailwind CSS v4.
* Colors (Defined in OKLCH for dynamic theming):
  - Primary Theme: Blue-to-Cyan gradient (135deg, #2563EB to #06B6D4).
  - Backgrounds: Light mode uses near-white `oklch(0.98 0.002 240)`, dark mode uses deep navy/black `oklch(0.12 0.02 260)`.
* Typography:
  - Heading Font: Space Grotesk (`var(--font-space-grotesk)`).
  - Body Font: Inter (`var(--font-inter)`).
  - Logo Font: Georgia (Serif).
* UI Patterns:
  - Generous border radii: `rounded-2xl`, `rounded-3xl`, and `rounded-full`.
  - Glassmorphism: Heavy use of `backdrop-blur-md`, `bg-background/50`.
  - Borders: Thin `border border-border/50` for structure.
  - Glow Effects: Large radial gradients behind elements to create ambient lighting (`blur-[120px]`).
  - Shadows: Soft, colored drop shadows on interactive elements (e.g., `shadow-primary/20`).
* Animation Framework: Framer Motion.
  - Page elements use staggered fade-ins and upward y-axis translations on scroll.
  - Buttons have scale effects on hover/active (`hover:scale-105 active:scale-95`).
